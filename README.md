# 三轮舵轮简易模型

## 功能介绍

该项目是基于 webots 平台进行的运动仿真。
在 webots 上获取键盘的按键，控制仿真模型的运动。

## 效果展示

**运动效果图：**

![效果图](https://s2.loli.net/2022/02/10/3LzgyG4AXlkactE.gif)

## 依赖工具及软件环境

`工具:`Webots

`软件环境:`Windows10

## 使用方法

按键|运动
-|-
W|前
S|后
D|左
A|右
Q/E|左右自旋
